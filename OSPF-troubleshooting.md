# OSPF Troubleshooting

## Common OSPF Neighbor Issues
- Area mismatch
- Authentication mismatch
- MTU mismatch
- Network type mismatch
- Hello/Dead timer mismatch

## Useful Commands
```bash
show ip ospf neighbor
show ip ospf interface
show ip route ospf
debug ip ospf adj
```

## Troubleshooting Steps
1. Verify interface status
2. Check OSPF area configuration
3. Validate timers and MTU
4. Confirm authentication settings
5. Review routing table
