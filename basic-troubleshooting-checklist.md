# Basic Network Troubleshooting Checklist

## Layer 1
- Verify cables
- Check interface status
- Confirm link lights

## Layer 2
- Validate VLAN assignment
- Verify trunk configuration
- Check MAC address table

## Layer 3
- Verify IP addressing
- Check routing table
- Test connectivity with ping/traceroute

## Security
- Review firewall rules
- Validate ACLs
- Confirm VPN tunnel status
